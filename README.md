### Running:

1. Clone this repository 

```bash
git clone https://github.com/Sberm/Thats-Not-A-Vid.git
```

2. Build the project

```bash
mkdir build
cd build
cmake ..
cmake --build .
./Thats-Not-A-Vid
```

3. enjoy!!
