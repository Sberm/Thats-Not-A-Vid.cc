### Running:

1. Clone this repository 

```bash
git clone https://github.com/Sberm/Thats-Not-A-Vid.git
```

2. Install opencv library

2. Build the project, and run

```bash
# install opencv and cmake
# on Ubuntu
sudo apt install libopencv-dev cmake

# compile
mkdir build
cd build
cmake ..
cmake --build .
./Thats-Not-A-Vid
```
